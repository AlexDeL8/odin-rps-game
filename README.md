## Technologies Used
- HTML, CSS, Flexbox, JavaScript

## Features
- Singe game, best of 9 (*first to 5 wins*)
- CPU plays with randomly generated picks
- Keyboard functionality (*r - rock, p - paper, s - scissor. Also left, up, right respectively work too*)
